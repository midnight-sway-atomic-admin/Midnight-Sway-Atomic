Thunar Preferences Configuration

1. Display Tab

   1.1 Window Settings

       - Use custom window decorations (only applied to new windows): Disabled
       - Use current folder icon as window icon: Disabled

   1.2 View Settings

       - View new folders using: Last Active View
       - Remember view settings for each folder: Enabled
       - Sort folders before files: Disabled
       - Show file size in binary format: Disabled

   1.3 Thumbnails

       - Show thumbnails: Always
       - Only show thumbnails for files smaller than: Unlimited
       - Draw frames around thumbnails: Disabled

   1.4 Icon View

       - Text beside icons: Disabled

   1.5 Date

       - Format: Thu 14 May 2026 02:24:12 PM EDT

3. Behavior Tab

   3.1 Navigation

       - Single click to activate items: Disabled
       - Double click to activate items: Enabled

   3.2 Tabs and Windows

       - Open folders in new tabs on middle click: Disabled
       - Open new thunar instances as tabs: Disabled
       - Show full directory paths in tab titles: Disabled
       - Restore tabs on startup: Disabled

   3.3 Context Menu

       - Show action to permanently delete files and folders: Disabled

   4.1 File Transfer

       - Transfer files in parallel: Local Files Only (Idle Devices)
       - Use intermediate file on copy: Never

   4.2 Search

       - Include subfolders: Always

   4.3 Folder Permissions

       - Default behavior when changing folder permissions: Ask every time

   4.4 Security

       - Execute shell scripts: Never

   4.5 Volume Management

       - Enable Volume Management: Disabled

       - New Tab: Ctrl+T

       - Close Window: Ctrl+Q
       - Close All Windows: Shift+Ctrl+W

       - Redo: Shift+Ctrl+Z
       - Preferences...: Unassigned (...)
