1. The Midnight Sway Atomic rice has four main colors: the background (#000000); the colors of code (defined in the foot.ini via ANSI lines); the main color of the fonts (#8A97BF); the color of borders and highlights and various icons (also mainly #8A97BF); and the colors of the wallpaper.

2. To change the colors of the rice, ideally you will select a wallpaper with two dominant colors: the main color of the background of the wallpaper, and the main color of the main object of the wallpaper. The first becomes the background color of the rice (i.e., it replaces #000000); the second becomes the main color of the font, highlights, icons, and borders (i.e., it replaces #8A97BF). And then these two colors are then used to find a harmonious set of colors for the ANSI lines defined in the foot.ini.

3. Changing the background color of the rice is easy; but changing the color of the font and icons and borders and highlights is tedious because control over these colors is spread out in very many places (you must change the config of rofi, Firefox, sway, thunar, waybar, etc.), and you must also create newly colored versions of the desktop folders, WiFi icons, Firefox UI buttons, and battery icon--all of which are hard-coded in #8A97BF.
4. One of the listed problems in problems-to-be-fixed.txt is a test of the color shims in userChrome.css and userContent.css. Such a test to change the color scheme of the rice has never been attempted via these shims.
